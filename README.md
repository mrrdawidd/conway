# Conway's game of life in C and Raylib


Move using RMB

Zoom using SCROLL WHEEL

Click Space to pause and R to randomise again

Click Z and X to change simulation speed

Click C and V to change randomiser density

Click 1-8 and SHIFT + 1-8 to change rules

Click F to erase everything

Use LMB to draw and SHIFT + LMB to erase

Click Q and E to change brush size
